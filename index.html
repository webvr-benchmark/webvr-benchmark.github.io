Abstract

Evaluating webpage recreation from videos rather than static screenshots.

Existing web-generation benchmarks are mostly built around text prompts or static screenshots. WebVR shifts the task to demonstration videos, which carry richer evidence about layout, transitions, motion continuity, and interaction flow. The benchmark contains 175 synthesized webpages from diverse categories and evaluates generated outputs with a fine-grained visual rubric covering aesthetics, structure, section layout, and motion. Across 19 models, WebVR shows that modern MLLMs can often recover global style cues, but still struggle to faithfully implement dynamic behavior and fine-grained webpage details.

Task

Video-conditioned webpage recreation

Each model receives a reference screen-recording video and a set of visual assets, then produces a standalone executable HTML page for comparison against the reference behavior.

Data

Controlled synthesis to reduce contamination

The benchmark is constructed through captioning, semantic re-theming, asset grounding, multi-model generation, and rubric-based filtering rather than directly crawling existing webpages.

Evaluation

Human-aligned visual rubrics

Rubrics are decomposed into atomic binary checks over Global Aesthetics, Navigation and Footer, Section-Specific Layouts, and Interaction and Motion.

Overview

Benchmark overview

Rubric dimensions

  • GA

    Global color mood, typography, and visual coherence.

  • NF

    Header, menu, footer, and persistent UI structure.

  • SSL

    Section-level hierarchy, spacing, alignment, and layout fidelity.

  • IM

    Hover effects, transitions, scroll-triggered behaviors, and motion quality.

Reliability

Judge reliability

Without visual rubric

Agreement with human experts stays compressed, ranging from 59.3% to 66.7% on average across different modalities.

With visual rubric

Joint code-and-video judging rises to 86.7% average agreement, and Kimi-K2.5 reaches 96.0% alignment with human preference.

Failure Modes

Representative failure cases

Representative failure types

  • Structural collapse: invalid or nearly blank renderings.
  • Semantic divergence: coherent pages that ignore the demonstrated layout and theme.
  • Asset misplacement: correct high-level structure but wrong images or spatial grounding.
